Viết code HTML cho View sử dụng với mục đích đăng nhập:
- Nhập Email
- Nhập mật khẩu
- Nút "Đăng Nhập"

Viết mã HTML cho View có chức năng đổi mật khẩu:
- Nhập mật khẩu cũ
- Nhập mật khẩu mới
- Xác nhận lại mật khẩu mới
- Nút "Đổi mật khẩu", link (dạng nút) để "Bỏ qua"

Viết code HTML cho view có chức năng tiềm kiếm và hiển thị danh sách nahf cung cấp Yêu Cầu:
- Phần cung cấp thông tin tìm kiếm bao gồm: ô để nhập tên nhà cung cấp cần tìm, sau ô là nút Search, 
sau nút search là link(dạng button) đến chức năng bổ sung(create).
- Thông báo số dòng dữ liệu tìm được
- Bảng hiển thị danh sách thông tin cần hiển thị: Tên nhà cung cấp, Tên giao dịch, Điện Thoại,Email,Địa chỉ,Tỉnh/Thành.
Tại ô cuối của mỗi dòng có link dạng nút đến chức năng Edit và Delete
- Thanh phân trang(pagingnation) được bố trí bên phải.


Viết code HTML cho view để nhập các dữ liệu sau:
- Tên nhà cung cấp
- Tên giao dịch
- Điện thoại
- email
- địa chỉ
- tỉnh thành(chọn từ select)

Viết code HTML cho chức năng xóa nhà cung cấp hiển thị thông tin nhà cung cấp cần xóa,
hiện xác nhận trước lúc xóa

(Tôi chọn cách bố trí giao diện của view hiển thị, tìm kiếm dữ liệu theo cách như đoạn mã html sau)
